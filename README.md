🕵️ The Last Clue
The Last Clue is an interactive detective mystery web game where you investigate a murder, interrogate suspects, and uncover hidden truths to identify the real killer.
Every clue matters. Every suspect is hiding something.

🎮 Game Concept
You play as a detective assigned to solve a complex murder case. Your job is to:
🔍 Explore clues
🧠 Interrogate suspects
📄 Analyze evidence
🎯 Find the real killer
But nothing is straightforward — some clues are misleading, and every suspect has secrets.
